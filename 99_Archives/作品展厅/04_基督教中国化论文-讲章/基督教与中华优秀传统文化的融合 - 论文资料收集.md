## 一、核心神学家观点（已整理）

### 1. 丁光训（1915-2012）
**身份**: 中国基督教三自爱国运动委员会名誉主席、中国基督教协会会长

**主要观点**:
- 提出"宇宙的基督"概念，强调基督是宇宙的主宰，不仅限于教会
- 主张基督教与中国文化对话，寻找共通点
- 强调基督教的社会责任和服务精神
- 支持三自爱国原则：自治、自养、自传

**代表著作**:
- 《丁光训文集》（译林出版社，2006年）
- 《基督教的中国化三要素》（金陵神学志，1999年）

**引用格式**:
```
[1] 丁光训. 丁光训文集[M]. 南京: 译林出版社, 2006.
[2] 丁光训. 基督教的中国化三要素[J]. 金陵神学志, 1999(1): 1-10.
```

### 2. 赵紫宸（1888-1979）
**身份**: 中国基督教神学家、教育家、诗人

**主要观点**:
- 主张基督教与儒家思想对话
- 提出"中国的基督教神学"概念
- 强调基督教伦理与儒家伦理的契合
- 主张基督教艺术的民族化

**代表著作**:
- 《基督教进解》（1947年）
- 《耶稣传》（1935年）
- 《中国基督教神学建设的任务》（神学志，1940年代）

**引用格式**:
```

[4] 赵紫宸. 耶稣传[M]. 上海: 青年协会书局, 1935.
```

### 3. 吴耀宗（1893-1979）
**身份**: 中国基督教三自爱国运动发起人、中国基督教协会首任会长

**主要观点**:
- 发起三自爱国运动（1950年）
- 主张基督教与社会主义社会相适应
- 强调基督教的爱国主义传统
- 支持基督教参与社会服务

**代表著作**:
- 《吴耀宗文集》（1990年代整理出版）
- 《三自革新运动的历史意义》（中国基督教三自爱国运动委员会，1950年代）

## 三、文化契合点研究

### 1. 儒家文化与基督教

**共通点**:
- 仁爱：儒家"仁者爱人"与基督教"爱人如己"
- 孝道：儒家"孝悌"与基督教"孝敬父母"
- 和谐：儒家"和为贵"与基督教"使人和睦"
- 修身：儒家"修身齐家"与基督教"圣灵果子"

**引用格式**:
```
[10] 牟宗三. 儒家思想与基督教精神[M]. 台北: 学生书局, 1985.
```

---

### 2. 道家文化与基督教

**共通点**:
- 谦卑：道家"上善若水"与基督教"虚心的人有福了"
- 自然：道家"道法自然"与基督教"神创造万物"
- 无为：道家"无为而治"与基督教"不要为明天忧虑"

---

### 3. 佛教文化与基督教

**共通点**:
- 慈悲：佛教"慈悲为怀"与基督教"神就是爱"
- 舍己：佛教"舍身饲虎"与基督教"基督舍命"
- 来世：佛教"轮回转世"与基督教"永生盼望"
